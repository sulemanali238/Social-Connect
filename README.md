# SocialConnect 📱

SocialConnect is an Android application designed to bridge social networking and seamless connectivity. Built using modern Android development practices, this project focuses on high performance, an intuitive user interface, and robust asset management.

---

## 🚀 Features

* **💬 Real-Time Chat:** Seamless, instant messaging capabilities allowing users to connect instantly.
* **📸 Profile Stories:** Share temporary visual updates with friends that disappear after 24 hours.
* **📝 Feed Posts:** Create, share, and interact with text and media posts on a dynamic social feed.
* **👤 User Profiles:** Fully customizable user profiles to display bio, profile pictures, and user activity.
* **🎨 Adaptive Icon Support:** Fully updated and optimized for Android 8.0 (API 26) and above, supporting standard, round, and monochrome app icons.
* **📱 Modern UI:** Clean, responsive design optimized for various screen sizes and DPI classifications.
